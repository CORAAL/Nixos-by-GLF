# Pré-requis

- Le gestionnaire de paquets `nix`
- `gnumake`

# Construire

1. Clonez le projet 

```bash
git clone https://github.com/GLF-OS/Nixos-by-GLF.git
```

2. Déplacez-vous dans l'arborescence

```bash
cd Nixos-by-GLF
```

3. Construire

```bash
make iso 
```
